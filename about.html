About This Site

This website is a personal space dedicated to hosting and sharing text-related work. It serves as a centralized repository for essays, articles, stories, and other written content.

Purpose

The purpose of this site is to provide a clean, distraction-free platform for reading and exploring written content. The design emphasizes readability and simplicity, allowing the text to take center stage.

Getting Started

To add new text works to this site:

  1. Create a new HTML file in the works/ directory
  2. Use the sample work as a template for formatting
  3. Add a link to your new work on the Works page
  4. Optionally feature it on the home page
